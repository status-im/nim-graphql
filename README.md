## Introduction

nim-graphql is Nim implementation of [GraphQL](https://spec.graphql.org/June2018/) specification and consist of
3 major parts:

* Parser
* Server
* Client

Expect nothing come soon because of quite low priority at the moment.

## Contributing

When submitting pull requests, please add test cases for any new features
or fixes and make sure `nimble test` is still able to execute the entire
test suite successfully.
